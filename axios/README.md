- 原生的方案来做网络请求
    - XMLHTTPRequest
    - fetch
- 在实际开发中，我们一般不会用原生的网络请求
    - 原生对要做简单对封装
    - 需要做额外的操作
        - 1、需要自己来封装原生
        - 2、原生某些功能不具备，需要自己来完成
            - 请求拦截；
            - 响应拦截；～ 可能获取到的某些数据不是我们想要的
        - 3、js运行环境：浏览器/Node
            - fetch（仅限于浏览器）；在Node环境中使用http
        - 使用axios可以解决上述问题；
